# OptativoIII2erParcial
Repositorio de entrega de srcipts del 2do parcial

Que tal profe?

En este repositorio se encuentran los scripts para la entrega del examen:
En el MAIN se enceuntra:
Program.cs : Main de la API
appsettings.json : Conexión a la DB mediante SQL SERVER, utilizo Microsoft.EntityFrameworkCore

- Dentro de la carpeta Models están los archivos .cs de cada tabla de la DB, con sus respectivos CRUD y sus controladores dentro del mismo script,
- Dentro de la carpet View se encuentran las vistas en Razor como endpoint para el usuario, dividida por carpetas para cada tabla, y dentro de las mismas están las pantallas para el CRUD.
